# JaQuan Brown Portfolio

This repository contains the GitHub Pages portfolio for **jmbrowntech**.

## Publish with GitHub Pages

1. Sign in to GitHub.
2. Create a new public repository named `jmbrowntech.github.io`.
3. Upload all files from this package into the root of that repository.
4. Open the repository's **Settings**.
5. Select **Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the `main` branch and `/root`.
8. Save.

Your website should then appear at:

`https://jmbrowntech.github.io`

## Before publishing

- Replace both LinkedIn placeholder links in `index.html` with your real LinkedIn profile URL.
- Add your resume PDF to the `assets` folder and name it exactly:
  `JaQuan_Brown_Resume.pdf`
- Review all wording and remove anything you do not want public.
- Do not upload confidential company information, patient information, internal screenshots, IP addresses, passwords, or private ticket details.

## Files

- `index.html` — website content
- `style.css` — bright, clean visual design
- `script.js` — automatic copyright year
- `assets/` — place your resume PDF here
